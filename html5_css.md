### Comment out HTML

```
<!--
<h1>Hello World</h1>
-->
<h2>CatPhotoApp</h2>
<!--

-->
```

### Use a CSS Class to Style an Element

```
<style>
  .red-text {
    color: red;
  }
</style>

<h2 class="red-text">CatPhotoApp</h2>

```

